# 蝶々反乱

札幌ゲームキャンプ2025で開催されたゲームジャムで制作されたゲームです。

札幌を拠点とするゲーム会社が合同で開催するイベント「札幌ゲームキャンプ2025」で、プロのゲームクリエイターによる講座やゲームジャムなど、ゲーム開発に関する様々なイベントが開催されました。

詳細はこちら：[札幌ゲームキャンプ公式Webサイト](https://sapporo-game-camp.com/2025/)

---

## チーム名

**アルティメット革命軍**

---

## ゲーム概要

敵を撃破することで、失われた自然が徐々に回復していきます。制限時間内に敵を倒してスコアを競い、ランキングに載ることを目指そう！

| 項目 | 内容 |
|------|------|
| ジャンル | 2Dアクション |
| プレイ人数 | 1人 |
| プレイ時間 | 5分程度 |
| 対応プラットフォーム | Windows |

---

## 操作方法

### キーボード

- **Wキー**：上
- **Aキー**：左
- **Sキー**：下
- **Dキー**：右
- **スペースキー**：弾を発射
- **ESCキー**：ポーズ

### ゲームコントローラー

- **左スティック / 十字キー**：移動
- **ボタン（右のボタン）、もしくはボタン（上のボタン）**：弾を発射
- **Optionボタン**：ポーズ

### ヒント

敵の攻撃パターンを見極めて、素早く回避しよう！
