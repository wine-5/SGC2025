#### 全体の領地をどれくらい占めているかを視覚的にわかりやすくするなにか
**実装: 円形ゲージ（Radial Gauge）**
- GroundManagerに`GetGreenificationRate()`追加（緑化率を計算）
- InGameUIに円形ゲージとパーセンテージテキストを追加
- UI Image（Type: Filled, Fill Method: Radial 360）を使用
- 緑化時にゲージが回転して埋まる演出
- 色: 緑系グラデーション（低: 薄緑 → 高: 濃緑）
- 位置: 画面右上（スコア・タイマーの近く）
